# 🐉 A Lenda do Meu Amor

Um cartão digital interativo, romântico e animado, criado em HTML, CSS e JavaScript com estética medieval/fantasia inspirada em reinos, dragões, castelos, espadas e pergaminhos.

O projeto foi desenvolvido como uma experiência especial de aniversário, com uma tela inicial temática, uma página intermediária de entrada no reino e uma mensagem romântica em formato visual cinematográfico.

https://anagoncalves-tech.github.io/famaro_2026/

## ✨ Funcionalidades

- Página inicial com tema dark fantasy
- Dragão em SVG com efeito visual
- Botões estilizados com estética medieval
- Página intermediária de transição
- Mensagem romântica em página separada
- Animação de brasas/fagulhas
- Efeito de revelação gradual da mensagem
- Corações animados
- Layout responsivo para celular e desktop

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- SVGs personalizados
- Google Fonts
